# MERIDIAN — Website Concepts

Six minimalist website directions for a **construction & architecture** company, built to a
simple, clean, image-led, low-motion brief (in the spirit of apple.com). Plus a shortlist of
six real, live reference sites that match the same aesthetic.

**Start here:** open [`index.html`](index.html) — a single gallery page that links to every
concept, grouped by category, with the reference sites at the bottom.

## The six concepts

| # | Concept | Category | Feel |
|---|---------|----------|------|
| 01 | [Product Showcase](concept-1-product/index.html) | Image-led | Apple-style, centered hero, huge whitespace |
| 02 | [Architectural Journal](concept-2-editorial/index.html) | Editorial | Serif display, asymmetric grid, warm paper |
| 03 | [Structured Grid](concept-3-swiss-grid/index.html) | Structured | Strict Swiss grid, numbered project register |
| 04 | [Cinematic (Dark)](concept-4-cinematic/index.html) | Image-led | Fullscreen photography, minimal overlaid text |
| 05 | [Studio Portfolio](concept-5-studio/index.html) | Editorial | Clean white, calm thumbnail grid |
| 06 | [Bold Typographic](concept-6-typographic/index.html) | Structured | Oversized monochrome type leads |

## Notes

- Each concept is a **self-contained HTML file** — no build step. Just open it in a browser.
- **MERIDIAN** is a placeholder brand used across all concepts so you compare *layout*, not content.
- Photography is from Unsplash (free to use), downloaded into [`assets/`](assets/) so the pages
  work offline. Replace with the client's real project photography later.
- Reference sites and rationale are in [`references.md`](references.md).
- Motion is deliberately minimal: a gentle fade-in on scroll, nothing more.
