# Reference Sites — Minimalist Construction & Architecture

Six real, live websites that already match the brief: simple, clean, mostly static
imagery, very little motion, Apple-like restraint. Use these to point at specific things
the client likes. All links verified working on 2026-06-23.

---

### 1. John Pawson — https://johnpawson.com
The reference point for architectural minimalism. Near-empty interface, enormous whitespace,
photography does all the talking. *Borrow:* the confidence to leave space empty.

### 2. TenBerke — https://tenberke.com
Extreme minimalism — almost no text on the homepage, full-screen architectural photography,
navigation tucked away. *Borrow:* image-first homepage, quiet nav. (Closest to Concept 4.)

### 3. David Chipperfield Architects — https://davidchipperfield.com
Calm, monumental, perfectly gridded. Restrained type, generous margins, no decoration.
*Borrow:* the structured project index and disciplined grid. (Closest to Concept 3.)

### 4. Vincent Van Duysen — https://vincentvanduysen.com
Warm minimalism — material, light and proportion foregrounded. Editorial, slow, tactile.
*Borrow:* the warm monochrome palette and large material close-ups. (Closest to Concept 2.)

### 5. Andersson-Wise Architects — https://anderssonwise.com
Clean white portfolio driven by high-quality project imagery on a calm grid.
*Borrow:* the thumbnail-grid portfolio layout. (Closest to Concept 5.)

### 6. Caruso St John Architects — https://carusostjohn.com
Quiet, typographic, almost academic. Structure and text carry the page over imagery.
*Borrow:* the typographic, low-ornament tone. (Closest to Concept 6.)

---

**Honourable mentions** (also live, also minimal): Perkins&Will
(https://www.perkinswill.com), 6a Architects (https://www.6a.co.uk),
Minale + Mann (https://minaleandmann.com).

**Common thread across all six:** big photography, tiny navigation, lots of empty space,
restrained type, and almost no animation beyond a gentle fade. Exactly the direction the
six concepts in this folder explore.
